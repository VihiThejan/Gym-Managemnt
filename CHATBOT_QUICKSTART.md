## Get Your Gemini API Key (2 minutes)

1. **Visit**: [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)
2. **Sign in** with Google
3. **Click** "Create API Key"
4. **Copy** the generated key

## Add API Key to Project

1. **Open** `megapower-react-app/.env`
2. **Replace** `YOUR_GEMINI_API_KEY_HERE` with your actual key:
   ```env
   REACT_APP_GEMINI_API_KEY=AIzaSyC_your_key_here
   ```
3. **Save** the file

## Start the App

```bash
# Terminal 1 - Backend
cd Backend
npm start

# Terminal 2 - Frontend
cd megapower-react-app
npm start
```

## Test the Chatbot

1. Open browser: `http://localhost:3000`
2. Click purple chat button (bottom-right corner)
3. Ask: "What are your membership packages?"
4. Get instant AI response! ðŸŽ‰
